## Learning Goals

### Frontend Development
- Understand the difference between frontend and backend development
- Learn HTML5 semantic structure and accessibility basics
- Master CSS concepts including Flexbox, Grid, positioning, and responsive design
- Build responsive user interfaces using Tailwind CSS
- Learn JavaScript fundamentals including variables, loops, functions, arrays, and objects
- Understand DOM manipulation and event handling
- Learn React fundamentals including components, props, state, hooks, and routing
- Build reusable and maintainable frontend components
- Manage application state using Context API
- Learn frontend testing basics using React Testing Library

### Apollo Client
- Set up Apollo Client in React and wrap the app with ApolloProvider
- Use useQuery for fetching data and useMutation for changing data.
- Send JWT token using Apollo Client headers.
- Understand Apollo cache basics and refetchQueries.


### Backend Development
- Understand how servers and APIs work
- Learn Python basics, functions, dictionaries, OOP and virtual environment.
- Learn HTTP methods, status codes, headers, and request/response lifecycle
- Learn FastAPI setup, routing, request body, path/query parameters and status codes.
- Understand JSON data structure and API communication
- Learn REST API concepts and GraphQL fundamentals
- Understand GraphQL vs REST and when GraphQL is useful.
- Build backend applications using FastAPI
- Create CRUD APIs using REST and GraphQL
- Understand authentication and authorization using JWT
- Learn API security and protected routes

### Database & Data Management
- Understand SQL vs NoSQL databases
- Learn PostgreSQL database fundamentals
- Design relational database schemas using primary and foreign keys
- Learn SQLAlchemy ORM for database interaction
- Perform CRUD operations with databases
- Manage user-specific and relational data

### Full Stack Integration
- Connect frontend and backend applications
- Integrate React frontend with GraphQL backend using Apollo Client
- Handle API requests, loading states, and error handling
- Build complete full stack task management applications

### DevOps & Deployment
- Learn Git and GitHub workflow
- Practice commits, branching, pull requests, and merge conflict resolution
- Understand Linux basics and terminal commands
- Learn Docker and Docker Compose basics
- Deploy frontend applications using Vercel or Netlify
- Deploy backend applications with production configuration

### Problem Solving & Professional Skills
- Improve debugging and troubleshooting skills
- Write clean, readable, and maintainable code
- Understand software development best practices
- Learn project organization and documentation
- Practice real-world development workflow
- Build confidence in presenting and explaining projects

### Final Objective
- Become capable of building and deploying production-ready full stack applications using:
  - React
  - Tailwind CSS
  - FastAPI
  - GraphQL
  - PostgreSQL
  - Docker
  - Apollo Client
  - Git & GitHub
