# One-Month Internship Report

## Introduction

During my first month as a Full Stack Developer Intern, I gained practical experience in both frontend and backend development. The training program was well organized and focused on building a strong foundation in modern web development technologies. Through daily assignments and projects, I learned how different parts of a web application work together, from designing user interfaces to developing APIs and managing databases.

The internship provided an opportunity to work on real development tasks, improve problem-solving skills, and understand the workflow followed in a professional software development environment.

---

## Work Completed During the First Month

### Frontend Development

During the frontend training period, I learned the fundamentals of web development and user interface design.

The tasks completed include:

* Created a GitHub repository and learned Git version control basics.
* Built a personal portfolio website using HTML.
* Styled webpages using CSS concepts such as selectors, box model, positioning, Flexbox, and Grid.
* Developed a responsive software company homepage.
* Created responsive login and registration pages using Tailwind CSS.
* Completed tasks covering HTML structure, CSS styling, responsive web design, and Tailwind CSS implementation.

---

### React Development

After learning JavaScript fundamentals, I moved to React development.

The tasks completed include:

* Set up React projects using Vite.
* Created a React Portfolio Homepage.
* Developed reusable Team Member Card components using props.
* Built a Product Quantity Counter using useState.
* Developed a React Registration Form with validation.
* Created a React Todo Application with add, delete, complete, and filter functionality.
* Implemented page navigation using React Router.
* Developed a User Dashboard by consuming API data.
* Managed authentication state using Context API.
* Completed a Task Management Frontend project.

---

### Backend Development

In the backend training phase, I learned Python, FastAPI, GraphQL, and database concepts.

The tasks completed include:

* Created Python programs such as:

  * Student Result Calculator
  * Number Guessing Game
  * Calculator Application
* Developed a class-based Employee Management System using Object-Oriented Programming concepts.
* Built Product APIs using FastAPI.
* Implemented CRUD operations using FastAPI.
* Created GraphQL schemas, queries, mutations, and resolvers.
* Designed database tables for a Task Management System.

* Performed database CRUD operations through FastAPI and GraphQL.

## What I Learned

One of the most valuable lessons during this month was understanding how frontend and backend systems work together to create a complete web application.

On the frontend side, I learned how to build responsive user interfaces, manage application state, handle user input, and consume APIs.

On the backend side, I learned how APIs are developed, how data is stored and retrieved from databases, and how GraphQL and REST APIs can be used to communicate between applications.

I also gained a better understanding of software development practices such as version control, code organization, debugging, and project structure.

## Challenges Faced

During the training period, I faced several challenges:

* Understanding React Hooks and component lifecycle concepts.
* Learning state management and Context API.
* Understanding asynchronous JavaScript concepts such as Promises and Async/Await.
* Connecting frontend applications with backend APIs.
* Understanding GraphQL schemas, queries, and resolvers.
* Configuring PostgreSQL database connections.
* Learning SQLAlchemy ORM and database sessions.
* Debugging application and API-related errors.

Although these challenges were difficult initially, solving them helped strengthen my technical knowledge and confidence.

The office environment was professional, friendly, and supportive. Team members were approachable and always willing to provide guidance whenever needed.
